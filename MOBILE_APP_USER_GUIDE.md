## Getting Started with Mobile

### Progressive Web App (PWA) Installation

#### Installing on iOS Devices

1. **Open Safari Browser**
   - Navigate to `https://your-cfmeu-domain.com`
   - Sign in with your CFMEU credentials

2. **Add to Home Screen**
   - Tap the **Share** icon (square with arrow) at the bottom of Safari
   - Scroll down and tap **"Add to Home Screen"**
   - Enter "CFMEU Organiser" as the name
   - Tap **"Add"** in the top right corner

3. **Verify Installation**
   - The CFMEU icon should appear on your home screen
   - Tap the icon to open the app in standalone mode
   - The app will open without browser bars for a native app experience

#### Installing on Android Devices

1. **Open Chrome Browser**
   - Navigate to `https://your-cfmeu-domain.com`
   - Sign in with your CFMEU credentials

2. **Install the App**
   - Look for the **"Install"** icon in the address bar
   - Tap **"Install"** or **"Add to Home Screen"**
   - Confirm installation by tapping **"Add"**

3. **Verify Installation**
   - The app icon appears in your app drawer
   - Launch the app for standalone experience
   - Grant necessary permissions when prompted

### First-Time Setup and Authentication

#### Initial Sign-In Process

1. **Launch the App**
   - Tap the CFMEU icon on your home screen
   - Allow the app to load (first load may take 30-60 seconds)

2. **Authentication**
   - Enter your CFMEU email and password
   - Enable **"Remember me"** for faster future access
   - Complete any two-factor authentication if required

3. **Permission Setup**
   The app will request several permissions:

   **Location Services**
   - Purpose: Find nearby projects, track site visits
   - Permission level: "While Using App" recommended
   - Essential for field organizers

   **Camera Access**
   - Purpose: Site photos, document scanning
   - Required for mapping workflows
   - Photos are stored securely in your project records

   **Photo Library Access**
   - Purpose: Upload existing photos and documents
   - Optional but recommended for documentation

   **Notifications**
   - Purpose: Task reminders, compliance alerts
   - Recommended for all field organizers

### "Closest to Me" GPS Search Feature

#### How GPS Search Works

1. **Location Detection**
   - The app automatically detects your current GPS location
   - Requires location permission (granted during setup)
   - Updates location every 30 seconds while moving

2. **Distance Calculation**
   - Calculates straight-line distance to each project
   - Considers current traffic conditions when available
   - Provides estimated travel time by car and walking

3. **Search Results**
   - Lists projects within your assigned patches
   - Shows distance and estimated travel time
   - Updates in real-time as you move

#### Using "Closest to Me"

1. **Access GPS Search**
   - From dashboard: Tap **"Find Nearby Projects"**
   - From map: Tap **"Use Current Location"** button
   - From projects list: Pull down to refresh with GPS

2. **Search Options**
   - **Radius**: Choose search distance (1km, 5km, 10km, 25km)
   - **Project Type**: Filter by construction type or value
   - **Compliance Status**: Show only projects needing attention

3. **Navigation Integration**
   - Tap any project for directions
   - Choose between Google Maps, Apple Maps, or Waze
   - One-tap navigation to project address

#### Best Practices for GPS Features

**Battery Optimization**
- Enable "Battery optimization" in device settings
- Close other GPS-heavy apps when using CFMEU app
- Consider bringing a portable battery pack for long days

**Location Accuracy**
- Wait 10-15 seconds after opening for GPS to stabilize
- Stand outside with clear view of sky for initial location lock
- Use "High accuracy" mode for precise site location

**Privacy Considerations**
- Location data is only used for work purposes
- Your movements are not tracked outside work hours
- Location history can be deleted in settings if needed

## Core Mobile Workflows

### Project Discovery and Navigation

#### Finding Projects Near You

1. **GPS-Based Discovery**
   - Open the app and allow location access
   - Dashboard will automatically show nearby projects
   - Tap **"View All Nearby Projects"** for complete list

2. **Map-Based Search**
   - Navigate to **Map** tab
   - Current location shown as blue dot
   - Project markers colored by compliance rating
   - Tap any marker for project details

3. **List-Based Search**
   - From **Projects** tab, tap **"Near Me"**
   - Results sorted by distance
   - Shows project name, address, and compliance status
   - Filter by project type or employer rating

#### Navigation to Project Sites

1. **Getting Directions**
   - From any project view, tap **"Get Directions"**
   - Choose your preferred navigation app
   - Confirm project address and start navigation

2. **Navigation Apps Integration**
   - **Google Maps**: Default for Android users
   - **Apple Maps**: Default for iOS users
   - **Waze**: Available for real-time traffic updates
   - Selection remembered for future use

3. **Arrival Confirmation**
   - When you arrive at the site, the app detects your location
   - Automatic check-in with GPS confirmation
   - Site visit log starts automatically

#### Site Arrival Protocols

1. **Safety First**
   - Before entering site, check safety requirements
   - Note personal protective equipment (PPE) needed
   - Look for site office for sign-in procedures

2. **Union Right of Entry**
   - Identify yourself as CFMEU organizer
   - Present union credentials if requested
   - Know your legal rights for site access

3. **Initial Site Assessment**
   - Take overview photos of the site
   - Note visible employer names and logos
   - Estimate workforce size from observation

### Mobile Mapping Workflows

#### Preparing for Site Mapping

1. **Equipment Check**
   - Ensure phone battery > 50%
   - Bring portable charger if possible
   - Clear phone storage for photos (minimum 1GB free)

2. **Information Gathering**
   - Review project details before arrival
   - Note any known employers or issues
   - Check last visit date and observations

3. **Approach Strategy**
   - Plan your entry point and first contact
   - Identify best times to approach workers
   - Consider safety briefings and break times

#### The Mobile Mapping Process

**Step 1: Start Mapping Workflow**

1. **Access Mapping**
   - From dashboard: Tap **"Start Mapping"**
   - From project: Tap **"Begin Mapping"**
   - From map: Tap project marker then **"Map This Site"**

2. **Project Verification**
   - Confirm project name and address
   - Note any discrepancies with records
   - Update project status if needed

3. **Initial Photos**
   - Capture site entrance and signage
   - Document workforce areas
   - Take overview shots of the entire site

**Step 2: Workforce Assessment**

1. **Total Worker Count**
   - Count all workers visible on site
   - Include all trades and roles
   - Estimate if exact count not possible
   - Note time of day for context

2. **Union Member Identification**
   - Identify union members through conversation
   - Look for union stickers or badges
   - Ask workers discretely about membership status
   - Document conversations respectfully

3. **Trade Breakdown**
   - Identify different trades on site
   - Count workers by trade type
   - Note which trades are unionized
   - Document any trade-specific issues

**Step 3: Employer Identification**

1. **Primary Contractor**
   - Identify main builder/developer
   - Look for site signage and offices
   - Ask site management for confirmation
   - Document ABN if available

2. **Subcontractors**
   - List all subcontractors on site
   - Note trade specialties
   - Estimate workforce sizes
   - Document contact information if available

3. **Employer Verification**
   - Take photos of company vehicles and equipment
   - Document employer logos and branding
   - Note any subcontractor relationships
   - Identify any sham contracting concerns

**Step 4: Union Delegate Mapping**

1. **Delegate Identification**
   - Ask workers about union representatives
   - Look for delegate badges or identification
   - Check with known union members
   - Document delegate contact details

2. **Delegate Assessment**
   - Assess delegate effectiveness
   - Note any training needs
   - Document safety representative roles
   - Record any delegate issues or concerns

3. **Relationship Building**
   - Introduce yourself to delegates
   - Offer support and resources
   - Schedule regular check-ins
   - Document communication preferences

**Step 5: Site Conditions Documentation**

1. **Access and Amenities**
   - Document site access procedures
   - Note parking availability
   - Record site hours and break times
   - Document available facilities

2. **Safety Assessment**
   - Note overall safety standards
   - Document any safety concerns
   - Record incident procedures
   - Identify safety representatives

3. **Photo Documentation**
   - Take systematic photos of:
     - Site entrance and signage
     - Workforce areas and facilities
     - Safety information boards
     - Amenities and break areas

#### Completing the Mapping Workflow

1. **Final Review**
   - Review all entered information
   - Ensure photos are properly labeled
   - Add any additional notes or observations
   - Check for completeness

2. **Submission Process**
   - If online: Tap **"Submit Mapping"**
   - If offline: Data saves automatically and syncs later
   - Receive confirmation of successful submission

3. **Follow-up Actions**
   - Schedule any required follow-up visits
   - Create tasks for compliance issues
   - Note any immediate actions needed
   - Update personal organizer notes

### Field Compliance Audits

#### Traffic Light Rating System

1. **Understanding the Ratings**
   - **Green**: Excellent compliance, strong union relationship
   - **Yellow**: Good performance with minor improvements needed
   - **Amber**: Moderate concerns requiring attention
   - **Red**: Serious compliance issues needing immediate action

2. **Assessment Categories**
   - **Union Respect**: Right of entry, delegate support, information access
   - **Safety Compliance**: Site safety, procedures, incident reporting
   - **Subcontractor Treatment**: Fair payment, sham contracting prevention
   - **EBA Compliance**: Current agreement, member coverage

3. **Rating Process**
   - Complete assessment based on site observation
   - Rate each category on 1-4 scale
   - Consider historical relationship and reputation
   - Document evidence supporting ratings

**Step 1: Access Compliance Audit**

1. **Starting the Audit**

2. **Delegate Support Assessment**
   - Evaluate how delegates are treated by management

   - Note training and development opportunities

3. **Information Access Review**
   - Assess willingness to share workplace information
   - Document information requests and responses

**Step 3: Safety Assessment**

1. **Site Safety Standards**
   - Observe overall safety culture and practices

2. **Safety Procedures Evaluation**
   - Review safety management systems

### Offline Data Collection and Sync

#### Understanding Offline Mode

1. **Automatic Offline Detection**
   - App detects poor or no internet connectivity
   - Automatically switches to offline mode
   - Continues full functionality with local data
   - Visual indicators show offline status

2. **Offline Capabilities**
   - Complete mapping workflows without connection
   - Access previously loaded project information
   - Take photos and collect evidence
   - Fill out all forms and assessments

3. **Data Storage**
   - All data saved locally on device
   - Secure encrypted storage protection
   - Automatic backup when connection restored
   - Multiple device sync capability

#### Working Offline Effectively

1. **Preparation for Offline Work**
   - Sync all relevant projects before going offline
   - Download maps for areas you'll be visiting
   - Ensure sufficient device storage space
   - Charge device fully or bring power bank

2. **Offline Data Collection**
   - Work normally - all features function offline
   - Data saves locally with timestamps
   - Photos stored locally and tagged
   - Forms completed and ready for submission

3. **Quality Assurance**
   - Review completed forms before going online
   - Ensure all required fields are completed
   - Verify photo attachments and labeling
   - Add any additional notes or context

#### Data Synchronization Process

1. **Automatic Sync**
   - When internet connection restored, automatic sync begins
   - All offline data uploaded to secure servers
   - Conflicts resolved with user prompts
   - Progress indicators show sync status

2. **Manual Sync Options**
   - Pull down on any list to trigger manual sync
   - Tap sync button in settings for immediate upload
   - Choose specific projects to sync first
   - Prioritize urgent compliance reports

3. **Sync Conflict Resolution**
   - If data modified online while offline, you'll see conflict options:
     - **Keep Offline Version**: Use your offline changes
     - **Use Online Version**: Discard offline changes
     - **Merge Changes**: Combine both versions when possible

4. **Sync Status Monitoring**
   - Check sync status in settings
   - Review failed sync items and retry
   - Monitor data usage during sync
   - Confirm all data successfully uploaded

## Field Optimizations

### Touch Interface Best Practices

#### Device Setup for Field Work

1. **Screen Protection**
   - Install tempered glass screen protector
   - Consider rugged case for construction site environments
   - Enable touch sensitivity settings if available
   - Clean screen regularly for optimal touch response

2. **Touch Optimization Settings**
   - Increase touch sensitivity in device accessibility settings
   - Enable "Touch and hold delay" adjustment for better control
   - Turn on "Touch accommodation" if you have motor difficulties
   - Adjust screen brightness for outdoor visibility

3. **Glove Compatibility**
   - Enable glove mode on supported devices
   - Consider stylus for precise input in cold weather
   - Test touch responsiveness with your work gloves
   - Practice common gestures with field equipment

#### Effective Touch Techniques

1. **Button Pressing**
   - Use thumb or index finger for consistent pressure
   - Ensure full finger contact with button surface
   - Hold briefly (1-2 seconds) for reliable response
   - Avoid tapping with fingernails or stylus tips

2. **Scrolling and Navigation**
   - Use smooth, steady gestures for scrolling
   - Practice pull-to-refresh for data updates
   - Master swipe gestures for navigation between sections
   - Use two-finger pinch for map zooming

3. **Text Input Optimization**
   - Use voice-to-text when possible for faster input
   - Enable predictive text and auto-correction
   - Practice with different keyboard layouts
   - Consider external keyboard for extensive data entry

#### Reducing Touch Fatigue

1. **Ergonomic Considerations**
   - Hold device at comfortable height and angle
   - Switch between hands during extended use
   - Take regular breaks from screen interaction
   - Use device stand for stationary work periods

2. **Gesture Shortcuts**
   - Learn double-tap shortcuts for common actions
   - Master long-press menus for additional options
   - Use swipe gestures instead of multiple taps
   - Customize gesture shortcuts when available

3. **Voice Integration**
   - Use voice commands for navigation
   - Dictate notes and observations hands-free
   - Enable voice control for accessibility
   - Practice voice commands in quiet environments first

### Battery Conservation Tips

#### Pre-Shift Battery Preparation

1. **Maximize Starting Charge**
   - Charge device to 100% before field work
   - Use high-quality charger and cable
   - Charge overnight to ensure full capacity
   - Consider removing case during charging for better heat dissipation

2. **Battery Health Optimization**
   - Enable optimized battery charging settings
   - Keep device updated with latest software
   - Avoid extreme temperatures (hot or cold)
   - Replace battery if health drops below 80%

3. **Power Bank Preparation**
   - Invest in high-capacity portable power bank (10,000mAh+)
   - Keep power bank charged and ready
   - Use short, high-quality charging cables
   - Test power bank output regularly

#### In-Field Battery Management

1. **Screen Optimization**
   - Reduce screen brightness to minimum usable level
   - Set shorter auto-lock timer (30 seconds recommended)
   - Enable dark mode to reduce power consumption
   - Turn off always-on display feature

2. **Connectivity Management**
   - Enable airplane mode in areas with poor reception
   - Turn off Bluetooth when not needed
   - Disable Wi-Fi when using cellular data
   - Limit background app refresh

3. **App-Specific Settings**
   - Close unnecessary background apps
   - Enable low power mode when battery < 50%
   - Reduce location accuracy to "battery saving"
   - Limit notification frequency and types

#### Emergency Power Strategies

1. **Power Conservation Mode**
   - Activate when battery < 20%
   - Disable all non-essential features
   - Focus on core mapping and audit functions
   - Preserve power for emergency communications

2. **Priority Operations**
   - Complete essential tasks first
   - Submit critical compliance reports immediately
   - Save progress frequently to prevent data loss
   - Use offline mode to reduce connectivity power drain

3. **Power Bank Usage**
   - Connect power bank before reaching critical battery level
   - Use power bank while continuing to work
   - Monitor charging progress and power bank capacity
   - Plan charging breaks during natural work pauses

### Working in Poor Connectivity

#### Connectivity Challenges

1. **Understanding Connection Types**
   - **5G/4G**: Best for urban areas with good tower coverage
   - **3G**: Slower but functional in many construction sites
   - **2G**: Basic connectivity in remote areas
   - **No Signal**: Complete offline operation required

2. **Signal Optimization**
   - Identify high-signal areas on site (higher elevation, open areas)
   - Move away from metal structures and concrete
   - Use signal booster if available and permitted
   - Note connectivity patterns for future visits

3. **Network Switching**
   - Enable automatic network switching in device settings
   - Manually switch between Wi-Fi and cellular when needed
   - Test different network types for best performance
   - Remember which network works best at each site

#### Offline-First Workflow

1. **Preparation Strategies**
   - Sync all relevant data before arriving at site
   - Download maps for planned work areas
   - Cache frequently accessed information
   - Prepare offline checklist of tasks

2. **Offline Data Collection**
   - Use fully featured offline mode
   - Collect all data normally, saves locally
   - Take photos with detailed descriptions
   - Note connectivity status for later reference

3. **Data Management**
   - Organize data by project and site visit
   - Use consistent naming conventions
   - Add detailed notes while memory is fresh
   - Mark priority items for immediate sync when online

#### Sync and Recovery Strategies

1. **Planned Sync Points**
   - Identify locations with good connectivity during travel
   - Schedule regular sync breaks throughout the day
   - Use home/office Wi-Fi for comprehensive sync
   - Prioritize urgent data sync over comprehensive updates

2. **Partial Recovery**
   - Sync individual projects when partial connectivity available
   - Use Wi-Fi at cafes/rest stops for opportunistic syncing
   - Enable background sync when good connection detected
   - Monitor sync progress and resolve conflicts promptly

3. **Data Verification**
   - Review synced data for completeness
   - Confirm all photos uploaded successfully
   - Check form submissions for processing
   - Verify offline data properly integrated with online records

   - Prepare appropriate protective equipment
   - Plan indoor work areas during severe weather
   - Have backup documentation methods (paper forms)

3. **Device Protection**
   - Use weather-resistant cases and screen protectors
   - Keep devices in protective bags when not in use
   - Allow devices to acclimate to temperature changes
   - Clean devices regularly to prevent dust accumulation

#### Working in Specific Conditions

## Troubleshooting Mobile Issues

### Sync Problems and Solutions

#### Common Sync Issues

1. **Sync Failure After Data Entry**
   - **Problem**: Data entered offline won't sync when online
   - **Causes**: Large file uploads, network timeout, server issues
   - **Solutions**:
     - Check internet connection strength
     - Retry sync with stable connection
     - Split large uploads into smaller batches
     - Clear app cache and retry

2. **Partial Sync Completion**
   - **Problem**: Some data syncs, other items remain pending
   - **Causes**: Network interruption, file size limits, server capacity
   - **Solutions**:
     - Review failed sync items individually
     - Prioritize important data for manual sync
     - Check available device storage
     - Restart app and retry sync

3. **Sync Conflicts**
   - **Problem**: Same data modified offline and online
   - **Causes**: Multiple device usage, collaborative editing
   - **Solutions**:
     - Choose appropriate conflict resolution option
     - Review conflicting changes carefully
     - Contact team members for coordination
     - Keep backup of important changes

#### Sync Optimization

1. **Connection Preparation**
   - Test internet speed before large syncs
   - Use Wi-Fi instead of cellular for large uploads
   - Ensure stable connection before starting sync
   - Plan sync during off-peak hours when possible

2. **Data Management**
   - Regularly delete unnecessary photos and files
   - Compress large images before uploading
   - Break large data entries into smaller chunks
   - Monitor storage usage and clean up regularly

3. **Preventative Measures**
   - Sync frequently to avoid large batch uploads
   - Test sync after major app updates
   - Keep app updated to latest version
   - Monitor sync status regularly

#### Advanced Sync Troubleshooting

1. **Manual Sync Procedures**
   - Force sync from settings menu
   - Use refresh-to-sync in individual screens
   - Restart device and retry sync
   - Reinstall app if persistent issues

2. **Data Recovery**
   - Check offline storage for unsynced data
   - Export critical data before troubleshooting
   - Contact support for data recovery assistance
   - Maintain regular backup practices

3. **Network-Specific Issues**
   - Test different network types (Wi-Fi, cellular)
   - Try different network providers
   - Use VPN if corporate network blocks sync
   - Report persistent network issues to IT support
