# CFMEU Mobile App User Guide

## Getting Started

### System Requirements

**Supported Devices:**
- iPhone 13 or newer (iOS 15+)
- Android phones with Android 10+
- Minimum 4GB RAM recommended
- 2GB free storage space

**Network Requirements:**
- 4G/5G mobile data or Wi-Fi
- GPS location services
- Camera access for photos
- Offline capability available

### Installation

**Progressive Web App (PWA):**
1. Open browser and navigate to your CFMEU platform URL
2. Tap "Share" button (iOS) or menu button (Android)
3. Select "Add to Home Screen" or "Install App"
4. Follow on-screen prompts to complete installation

**Benefits of PWA Installation:**
- Works offline with cached data
- Faster loading and performance
- Push notifications for important updates
- Native app-like experience
- Automatic updates

### First Login

1. **Authentication**
   - Enter your CFMEU email and password
   - Two-factor authentication if enabled
   - Accept terms of service and privacy policy

2. **Profile Setup**
   - Verify your contact information
   - Set your notification preferences
   - Configure your home location for GPS features

## Project Discovery and Navigation

### Finding Projects

**"Closest to Me" Feature:**
- Automatically detects your current location
- Sorts projects by distance
- Provides estimated travel time
- Updates as you move between sites

**Map Discovery:**
- Interactive map showing all projects
- Color-coded by compliance status
- Tap for project details and directions
- Filter by patch or employer

**Search Options:**
- Project name or address search
- Builder/employer filter
- Status and stage filtering
- Recent search history

### GPS and Navigation

**Location Services:**
- Enable GPS for accurate distance calculations
- "Navigate" button opens Google Maps or Apple Maps
- Turn-by-turn directions to project sites
- Location history for visit reporting

**Privacy Settings:**
- Location used only for project distance calculations
- No tracking of movements between sites
- Can disable GPS and use manual project selection
- Location data deleted after 30 days

**Offline Maps:**
- Download map areas for offline use
- Cached project locations and boundaries
- Works without internet connection
- Syncs when connection restored

## Site Visit Workflow

### Site Visit Wizard

**Starting a Visit:**
1. Select target project from dashboard
2. Tap "Start Site Visit" button
3. Confirm project details and location
4. Proceed through step-by-step workflow

**Wizard Steps:**
1. **Project Confirmation**
   - Verify project details
   - Record visit date/time
   - Note site conditions

2. **Contact Mapping**
   - Add new contacts discovered
   - Update existing contact information
   - Record union membership status

3. **Compliance Assessment**
   - Traffic light compliance ratings
   - Safety observations
   - Facility conditions review

4. **Workforce Documentation**
   - Total workers count
   - Union member percentage
   - Subcontractor presence

5. **Photo Documentation**
   - Site entrance photos
   - Safety signage
   - Compliance issues
   - Union delegate presence

**Working Without Connection:**

**Data Sync Process:**

**Offline Indicators:**

## Mobile Ratings System

### Employer Rating Wizard

**Accessing Ratings:**
- Navigate to employer profile
- Tap "Rate Employer" button
- Select appropriate rating track

**Rating Tracks:**
1. **Project Data Track**
   - Based on formal project records
   - Compliance reports and documentation
   - Documented incidents and issues
   - Payment history and EBA status

2. **Organiser Expertise Track**
   - Personal experience with employer
   - Site observations and interactions
   - Relationship quality assessment
   - Professional judgment and expertise

3. **Worker Feedback Track**
   - Reports from union members
   - Delegate observations and feedback
   - Worker complaints and concerns
   - Anonymous member surveys

**Rating Categories:**
- **Union Respect** (4-point assessment)
- **Safety Standards** (Traffic light system)
- **Sham Contracting** (Risk level assessment)
- **Wage Compliance** (Payment history)
- **Training Investment** (Apprentice ratios)
- **Job Security** (Employment stability)

### Confidence Scoring

**Understanding Confidence Levels:**
- **Very High (95%)**: Direct personal experience, multiple recent interactions, verified information
- **High (80%)**: Regular contact, solid understanding, some direct experience
- **Medium (60%)**: Some experience, mixed observations, partial information
- **Low (40%)**: Limited information, hearsay, outdated knowledge

**Impact on Ratings:**
- Higher confidence ratings carry more weight
- Low confidence ratings require additional verification
- Confidence affects sorting and filtering
- System tracks confidence over time

## Troubleshooting and Support

### Common Issues

**Sync Problems:**
- Check internet connection
- Verify login credentials
- Clear app cache and data
- Restart the application
- Contact support if persistent

**GPS Issues:**
- Enable location services in device settings
- Check app location permissions
- Calibrate device compass
- Try outdoor location for better signal
- Use manual project selection as backup

**Battery Optimization:**
- Enable battery saver mode

- Use Wi-Fi when available
- Carry portable power source
